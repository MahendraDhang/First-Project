   #Amazon Clone Website (HTML + CSS)
This project is a front-end clone of the Amazon homepage created using HTML5 and CSS3.
The main purpose of this project is to practice responsive layout design, modern UI styling, and page structuring using pure HTML and CSS.

   #Technologies Used
HTML5
CSS3
Font Awesome Icons
Flexbox Layout System
